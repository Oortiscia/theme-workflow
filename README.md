## just some custom themes.

# kitty:
go to `.config/kitty/kitty.conf` on your system.
scroll to the near end of that file and comment out the line :

`include current-theme.conf`

then include the file path e.g.

`include documents/cottonBlackberry-kitty`


# telegram
- you can simply click the shared links. or:

download the theme. go to `chat settings` top right corner, `create new theme` and then import the file.

you don't need to unzip for tg.
